1. Project Title
E-Commerce Website with Admin Panel
________________________________________
2. Project Overview
This project is a modern e-commerce website that allows users to browse products, view details, add items to cart, and purchase products online. The system includes a powerful admin panel that helps manage products, users, orders, and website content.
The website focuses on providing a simple, attractive, and user-friendly shopping experience.

3. User Roles
3.1 Customer (User)
Customer can browse and purchase products.
3.2 Admin
Admin controls and manages the entire website and data.
